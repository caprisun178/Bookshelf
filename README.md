# Project Title
Bookshelf

## Project Overview
My app will act like a personal bookshelf for users. They can log what books they have read (as well as add personal notes about the story) as well as make lists of books to read. 

## What problem does it solve or what purpose does it serve? Who is the intended user?
-Most book readers like treat it like an accomplishment when they have read another book and it is very rewarding to display all the books they have read. Though books take up space that 
cannot always be available. So my application will act like a digital bookshelf for users to browse new reads and reflect on previous ones.

## Core Features

- Create/Edit/Delete Book records
    - Log books to read
    - log books that have been read
    - log personal notes
- Search books by genre and author
- Filter books by have read and to read
- View books covers and details via [Open Library]https://openlibrary.org/developers/api
- Login/logout functionality
- Responsive design for desktop

**Frontend:** React
**Backend:** Express
**Database:** PostgreSQL
**APIs:** [Open Library]https://openlibrary.org/developers/api
**Deployment:** Vercel, Render, or other


## Stretch Goals (Optional)
 - I would like to make it where you can have book clubs that are chat rooms where users can discuss books together
 - It would be cool to make this a social media site as well, where you can add friends as well as send recommendations
 - A rating system for users to rate the book
